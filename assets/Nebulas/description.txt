Description:
Nebulas is an autonomous metanet. Hyper-mapped Structural Metadata can handle increasingly complex on-chain data and describe these interactions; new consensus incentives provide lasting positive incentive; new upgrade capabilities to upgrade without hard forks. NAS was developed as an ERC20 token, but is now a Mainnet NRC20 token that serves as currency on the Nebulas ecosystem.
